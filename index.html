Responsive Web Design

Responsive web design is about creating web pages that look good on all devices, A responsive web design will automatically adjust for different screen sizes. A Responsive web design is also said to be a web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it.
